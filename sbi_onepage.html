PROBLEM

자산 규모 1위 저축은행의
디지털 접점을 정리했습니다.

PC 홈페이지와 사이다뱅크 앱을 하나의 디자인 언어로 통합하고,주요 사용 흐름을 함께 재설계했습니다.

BEFORE — 기존 문제 babilloan.com · sbisb.co.kr · m.sbisb.co.kr
  • 브랜드 파편화 — SBI저축은행, 바빌론(대출), 사이다뱅크(앱)가 각각 다른 사이트. 같은 회사인지 인지하기 어려움
  • 복잡한 정보구조 — 메뉴 depth 4단계 이상. 원하는 기능까지 클릭 5회 이상
  • 레거시 UI — EUC-KR 기반 2010년대 디자인. 토스·카카오뱅크 대비 신뢰감 격차 큼
  • PC/모바일 분리 — 별도 도메인 운영. 디자인 시스템 부재. 일관된 경험 없음
AFTER — 리디자인 목표 sbisb.co.kr (PC) + 사이다뱅크 (Mobile)
  • 통합 브랜드 경험 — PC와 모바일에서 동일한 톤&매너. 바빌론·사이다뱅크를
    홈에서 함께 안내
  • 단순화된 IA — GNB 4개 + 빠른서비스 바로 주요 6개 기능 1클릭 접근
  • 현대적 비주얼 — Pretendard 타이포, SBI 블루 컬러 팔레트, 카드 기반 UI
  • 디자인 시스템 구축 — Foundation부터 컴포넌트까지 체계적으로 정의

로그인 전/후 홈 분리

비로그인 상태엔 상품·공지 중심. 로그인 후엔 내 자산 요약과 맞춤 서비스를 즉시 표시합니다.

자산 3분할 요약

입출금·예금·대출 잔여금을 3개 카드로 한눈에 파악. 대출 잔여금은 빨간색으로 시각적 경고를 줍니다.

개인화된 퀵서비스

로그인 후에는 즉시이체·거래내역·대출조회 등 실사용 기능 중심으로 퀵서비스 바가 재구성됩니다.
